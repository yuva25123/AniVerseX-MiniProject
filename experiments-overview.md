# Experiment Classification

The project has been reorganized into experiment-wise folders without removing the original module-based structure.

## Folder Mapping

- `exp1` - Client-side form design and validation
- `exp2` - JavaScript events, validation, and objects
- `exp3` - jQuery and AJAX
- `exp4` - React application
- `exp5` - Spring Boot MVC CRUD with MySQL and JSP
- `exp6` - Node.js and Express multi-tier backend
- `exp7` - XML DOM processing

## Notes

- The original folders such as `frontend/`, `backend-node/`, `backend-springboot/`, and `xml-module/` are still present.
- The `exp1` to `exp7` folders are a classified view for submission and easier presentation.
