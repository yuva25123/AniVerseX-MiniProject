AniVerseX

AniVerseX is a combined Web Programming mini project that unifies the lab experiments into one anime recommendation and review application. The project is organized into separate but connected modules so each experiment requirement can be demonstrated clearly.

HTML Form Validation JavaScript Events and Objects GET / POST jQuery + AJAX React Spring Boot MVC Node.js + Express XML DOM
